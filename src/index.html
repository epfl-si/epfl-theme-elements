Getting Started

Get started with Web2018 for building responsive, mobile-first sites. Be sure to have your pages set up with the latest design and development standards. That means using an HTML5 doctype and including a viewport meta tag for proper responsive behaviors. Put it all together and your pages should look like this: 

<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

    <link rel="shortcut icon" type="image/x-icon" href="https://web2018.epfl.ch/{{VERSION}}/icons/favicon.ico">
    <link rel="icon" type="image/png" sizes="16x16" href="https://web2018.epfl.ch/{{VERSION}}/icons/favicon-16.png">
    <link rel="icon" type="image/png" sizes="32x32" href="https://web2018.epfl.ch/{{VERSION}}/icons/favicon-32.png">
    <link rel="apple-touch-icon" sizes="180x180" href="https://web2018.epfl.ch/{{VERSION}}/icons/apple-touch-icon.png">

    <link rel="stylesheet" href="https://web2018.epfl.ch/{{VERSION}}/css/elements.min.css">

    <title>Hello, world!</title>
  </head>
  <body>
    <h1>Hello, world!</h1>

    <script>
      svgPath = 'https://web2018.epfl.ch/{{VERSION}}/icons/icons.svg';
    </script>

    <script src="https://web2018.epfl.ch/{{VERSION}}/js/elements.min.js"></script>
  </body>
</html>

That's all you need for overall page requirements.

Web2018 CDN

The domain web2018.epfl.ch deliver cached version of compiled CSS, JS and icons to your project. 

<link rel="stylesheet" href="https://web2018.epfl.ch/{{VERSION}}/css/elements.min.css">
<script src="https://web2018.epfl.ch/{{VERSION}}/js/elements.min.js"></script>

<img src="https://web2018.epfl.ch/{{VERSION}}/icons/epfl-logo.svg" alt="Logo EPFL" class="img-fluid">

Make sure to include the auto-updating html parts (includes) into your project available at https://web2018.epfl.ch/download-me.html.

As most templating languages does not support including content from a remote server, you can retrieve all the includes via: 

wget -P /var/www/mywebsite/header -N -nd -r -l 1 https://web2018.epfl.ch/download-me.html

An example page with all the includes is available here.

Compiled CSS and JS

Download ready-to-use compiled code for Web2018 v{{VERSION}} to easily drop into your project, which includes:

  • Compiled and minified CSS
  • Compiled and minified JavaScript plugins
  • Icons and logo

This doesn't include documentation, source files, or includes.

Download

npm

Install Web2018 in your Node.js powered apps with the npm package: 

npm install epfl-theme-elements
